[Trait](http://php.net/manual/zh/language.oop5.traits.php)

> trait实现了代码的复用

> 突破了单继承的限制

> trait是类不是类 不能实例化

```1
class Demo 
{
	public function hello()
	{
		return __METHOD__;
	}
}
$obj = new Demo;
echo $obj->hello();		Demo::hello
```

```代码复用多继承
// 代码复用 多继承
trait Demo1
{
	public function hello1()
	{
		return __METHOD__;
	}
}
trait Demo2 
{
	public function hello2()
	{
		return __METHOD__;
	}
}
class Demo 
{
	use Demo1,Demo2;
	public function hello()
	{
		return __METHOD__;
	}
	public function test1()
	{
		return $this->hello1();
	}
	public function test2()
	{
		return $this->hello2();
	}
}
$obj = new Demo;
echo $obj->hello();		Demo::hello
echo "<hr />";
echo $obj->test1();		Demo1::hello1
echo "<hr />";
echo $obj->test2();		Demo2::hello2
```

```3
// 优先级	Demo>Test>Demo1
trait Demo1
{
	public function hello()
	{
		return __METHOD__;
	}
}
class Test
{
	public function hello()
	{
		return __METHOD__;
	}
}
class Demo extends Test
{
	use Demo1;
	public function hello()
	{
		return __METHOD__;
	}
}
$obj = new Demo;
echo $obj->hello();		
Demo::hello > Demo1::hello > Test::hello
```

```4
// 解决trait的小冲突
trait Demo1
{
	public function hello()
	{
		return __METHOD__;
	}
}
trait Demo2
{
	public function hello()
	{
		return __METHOD__;
	}
}
class Test
{
	public function hello()
	{
		return __METHOD__;
	}
}
class Demo extends Test
{
	use Demo1,Demo2
	{
		Demo1::hello insteadof Demo2;
		Demo2::hello as Demo2hello;
	}
	public function hello()
	{
		return __METHOD__;
	}
	public function test2()
	{
		return $this->Demo2hello();
	}
}
$obj = new Demo;
echo $obj->hello();		Demo::hello > Demo1::hello
echo "<hr />";
echo $obj->test2();		Demo2::hello2
```
